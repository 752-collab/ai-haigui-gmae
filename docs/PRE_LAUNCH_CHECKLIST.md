# 上线前最终检查清单

## 构建与质量

- [ ] 前端生产构建通过：`web` 目录 `npm run build`，无 TypeScript / ESLint 红线。
- [ ] 自动化测试通过：`web` `npm test`；若改动后端则 `backend` 侧测试或手工冒烟通过。

## 配置与密钥

- [ ] 生产环境 `VITE_API_BASE` 指向自有海龟汤后端域名（非模型商裸域名），CORS 与 HTTPS 已核实。
- [ ] AI 密钥与数据库连接串仅存在于部署密钥管理 / 环境变量，未写入仓库与日志。
- [ ] `.env` / `.env.local` 未误提交；`.env.example` 与文档中的变量名一致。

## 对局与合规（AGENTS / PRD）

- [ ] 提问上限：第 20 问后无法再提问（服务端拒绝为准）。
- [ ] 通关当回合不插入阶段提示；6 / 10 / 15 提示锁位与顺序正确。
- [ ] 接口与日志不泄露汤底全文或可逆推断字段；Reveal 页展示为落库原文。

## 用户体验与稳定性

- [ ] 弱网/超时：对局不卡死，有明确错误提示，计数与状态与后端一致。
- [ ] 刷新/重进：session 恢复或大厅新开行为符合产品设计；隐私模式下用户知晓进度可能不保存。
- [ ] 主流浏览器与目标设备分辨率下布局与可访问性（焦点、读屏关键文案）可接受。

## 发布与回滚

- [ ] 版本号或构建标识可追踪；若有后端迁移，迁移已在预发执行并校验。
- [ ] 回滚步骤与监控告警联系人已确认。
